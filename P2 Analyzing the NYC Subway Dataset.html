Choose statistical test¶

From the graph we can see that both distribution are right-skewed, so we could not use z-test or t-test. We also don't know distribution of the entries population for rainy days and non-raniy days. The Mann Whitney U test is a non-parametric test that can be used to test, for two populations with unknown distributions. So Mann Whitney U test would be appropriate for our choice. I would prefer to use two-tailed test.

Hypothesis¶

Giving random draws x from entries population in rainy days, y from entries population in non-rainy days, we have below null hypothesis and alternative hypothesis

$$H_0: P(x > y) = 0.5$$


$$H_a: P ( x > y ) \neq 0.5$$

Calculate p value¶

p value for two-tailed test is 5.48213914249e-06

mean entries in rainy days is 2028.19603547
mean entries in non-rainy days is 1845.53943866

Significance and interpretation¶

We use two-tailed test and get p value $p=5.48\times 10^{-6},\; p << 0.01$, so we reject the null hypothesis, thus we accept the alternative hypothesis $P ( x > y ) \neq 0.5$, in other words probability of entries in rainy days larger than enties in non-rainy days is not equal to 0.5. This means that rain do affect subway entries.

Section 2. Linear Regression¶

I use OLS for linear regression, and try to add information to features as more as possible, so I would add all numeric variables about the weather, they are precipi, pressurei, tempi, wspdi, meanprecipi, meanpressurei, meantempi, meanwspdi. For dummy variables, UNIT and station are similar, and I find that only use UNIT as feature would result in $R^2$ 0.375, only use station as feature would result in $R^2$ 0.324, so I prefer to add UNIT as one of the dummy variables. For dummy variables about dates, datetime's information already cover DATEn, TIMEn, hour, day_week, weekday, so I would only add datetime as one of dummy variables. Finally I would add dummy variable conds, because adding it increases $R^2$ for my model. For weather_lat and weather_lon, I don't think it would affect entries, and acturally I also try to add these, but it does not improve $R^2$, so I would not contain it to my final model.

import statsmodels.api as sm 

def linear_regression(features, values):
    """
    Perform linear regression given a data set with an arbitrary number of features.
    """
    features = sm.add_constant(features)
    model = sm.OLS(values, features)
    results = model.fit()
    return results

values = turnstile_weather['ENTRIESn_hourly']

features = pd.get_dummies(turnstile_weather['UNIT'], prefix='unit')
